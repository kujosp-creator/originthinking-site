# Security Policy

## Supported Versions
This repository hosts a static website deployed via Cloudflare Pages.
There is no backend runtime or database.

## Reporting a Vulnerability
If you believe you have found a security issue, please report it responsibly.

Contact:
hello@originthinking.com

Please include:
- Description of the issue
- Steps to reproduce (if applicable)
- Affected URLs or files
- Screenshots or logs if available

## Scope
In scope:
- Client-side vulnerabilities (XSS, injected scripts)
- Misconfigured headers
- Asset exposure beyond intended public access

Out of scope:
- DDoS attacks (handled by Cloudflare)
- Issues requiring authenticated backend access
